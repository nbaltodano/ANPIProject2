Antes de compilar se requiere tener instalado:

    CMake	3.10.2
    g++	7.3
    boost	1.65.1
    GNU/Octave	4.2.2
    OpenCV	3.1
  
Para instalar:

    Ir al directorio
    
Ejecutar en consola:
  
  
    cmake ../ -DCMAKE_BUILD_TYPE=Debug
    make

Para ejecutar ir a la carpeta bin y ejecutar:
    
    ./proyecto2
    ./test
    ./benchmark
